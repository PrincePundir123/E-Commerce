# 🛒 E-Commerce Web Application (Amazon Clone)

A responsive e-commerce front-end application that simulates an Amazon-like shopping experience with product browsing, cart management, checkout, and order tracking.

---

## 🌍 Overview

This project is a client-side shopping platform built using **HTML, CSS, and JavaScript**.  
It dynamically loads product data and allows users to browse items, manage a cart, and simulate order checkout.

---

## 🚀 Features

- 🛒 Add, remove, and update items in cart
- 💳 Checkout with price summary
- 📦 Order tracking pages
- ⚡ Dynamic product loading using Fetch API
- 🧪 Unit testing using Jasmine
- 💾 Cart data stored in LocalStorage

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3 (Flexbox & Grid)**
- **JavaScript (ES6+)**
- **Jasmine**
- **Fetch API**
- **LocalStorage**

---

## ⚙️ How It Works

1. **Product Loading**
   - Products are fetched from a JSON file using `fetch()` and `async/await`.

2. **Cart System**
   - Cart data is stored in a JavaScript array and saved in **LocalStorage**.

3. **Checkout**
   - The checkout page reads cart data and calculates:
   - total price
   - shipping
   - taxes
